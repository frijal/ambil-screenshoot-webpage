Samba adalah solusi standar industri di Linux untuk berbagi folder dan printer dalam jaringan lokal agar bisa diakses oleh komputer Windows, macOS, dan Linux lainnya. Dengan tool grafis seperti system-config-samba, kita bisa mengatur folder bersama tanpa harus menyentuh file konfigurasi yang rumit secara manual.
