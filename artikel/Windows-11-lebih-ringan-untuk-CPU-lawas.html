Membuat Windows 11 Lebih Ringan untuk CPU Lawas

Windows 11 membawa banyak pembaruan visual dan fungsional, namun bisa terasa berat pada komputer dengan prosesor (CPU) generasi lama. Berikut adalah beberapa langkah optimasi yang bisa Anda lakukan agar performa Windows 11 menjadi lebih responsif.

Langkah-langkah Optimasi

  • Matikan Efek Visual: Nonaktifkan animasi dan transparansi yang tidak perlu. Buka Settings > Accessibility > Visual effects dan matikan "Transparency effects" serta "Animation effects".
  • Nonaktifkan Startup Program: Kurangi program yang berjalan otomatis saat booting. Buka Task Manager (Ctrl+Shift+Esc) > Startup apps dan nonaktifkan aplikasi yang tidak penting.
  • Gunakan Storage Sense: Bersihkan file sementara dan cache secara otomatis. Buka Settings > System > Storage dan aktifkan "Storage Sense".
  • Update Driver: Pastikan semua driver, terutama driver kartu grafis, selalu diperbarui ke versi terbaru.
  • Gunakan Antivirus Ringan: Hindari antivirus pihak ketiga yang berat. Windows Defender bawaan sudah cukup baik dan terintegrasi dengan sistem.
