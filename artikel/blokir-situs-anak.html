ğŸ§±Blokir di Level Router

Saran paling populer; akses pengaturan router untuk blacklist situs, gunakan fitur parental control bawaan (misalnya ASUS routers), atau tambah ban list. Ini blokir semua perangkat di jaringan rumah.

Kenapa ini terbaik? Karena gak cuma komputer anak yang kena, tapi juga HP, tablet, atau TV pintar. Anak gak bisa bypass pake perangkat lain. Coba masuk ke alamat router (biasanya 192.168.1.1 atau 192.168.0.1), login pake username/password default (lihat di belakang router), terus cari menu Parental Control atau URL Filtering. Tambahin situs kayak contoh-situs-berbahaya.com ke daftar blokir. Selesai!

ğŸ“Edit File Hosts

Ubah /etc/hosts di Linux untuk redirect situs berbahaya ke localhost (127.0.0.1), sehingga situs "hilang" dari akses. Sederhana tapi bisa dibypass anak pintar.

Cara gampang: Buka terminal, ketik sudo nano /etc/hosts, tambahin baris kayak gini:

127.0.0.1    facebook.com
127.0.0.1    tiktok.com

Simpan, restart browser. Situs langsung gak bisa dibuka! Tapi ingat, anak yang udah tahu Linux bisa edit balik. Jadi, kasih password kuat di akun admin.

ğŸŒGunakan DNS Family-Friendly

Ganti DNS ke layanan gratis seperti OpenDNS (Family Shield/Home), AdGuard DNS (family-dns.adguard.net), CleanBrowsing, atau NextDNS. Ini filter konten dewasa secara otomatis di seluruh jaringan, dengan laporan real-time.

Pilihan DNS terbaik 2025:

  • OpenDNS Family Shield: 208.67.222.123 dan 208.67.220.123
  • AdGuard Family: family-dns.adguard.net
  • NextDNS: Bikin akun gratis, custom denylist
  • Cloudflare Family: 1.1.1.3 dan 1.0.0.3

Ganti DNS di router biar semua perangkat kena. Atau di komputer anak aja kalau mau spesifik.

  • Install Timekpr-nExT untuk batasi waktu penggunaan komputer berdasarkan akun anak.
  • MintNanny atau Malcontent untuk blokir domain spesifik dan monitor penggunaan (tersedia di repo Linux Mint).
  • GNOME Parental Controls untuk desktop environment GNOME.
  • Pi-hole atau AdGuard Home untuk network-wide filtering (jalankan di Raspberry Pi).

Cara install Pi-hole: Butuh Raspberry Pi atau PC lama. Ikutin tutorial resmi di pi-hole.net. Setelah jalan, semua iklan dan situs berbahaya otomatis diblokir di seluruh rumah!

ğŸ”¥Proxy dan Firewall Advanced

Gunakan Squid Proxy atau iptables untuk redirect traffic HTTP/HTTPS ke proxy internal yang filter situs. Cocok untuk user tech-savvy.

Contoh perintah iptables:

sudo iptables -t nat -A PREROUTING -i eth0 -p tcp --dport 80 -j REDIRECT --to-port 3128

Tapi ini buat yang udah jago Linux. Kalau baru belajar, mending pake Pi-hole aja.

Meski Harry anti-Google, disebutkan parental control di Google Family Link untuk blokir situs berdasarkan usia. Alternatif: Proton VPN di Chrome untuk tambah lapisan privasi.

âœ…Whitelist vs Blacklist

Lebih baik whitelist (hanya izinkan situs aman) daripada blacklist, terutama untuk anak kecil, agar gak ada celah.

Contoh: Izinkan cuma google.com, khanacademy.org, youtube.com/kids. Sisanya otomatis diblokir. Aman banget!

Anak pintar bisa bypass (misalnya via flip phone atau akses fisik), jadi kombinasikan dengan pendidikan moral. Beberal saran ekstrem seperti cabut hardware internet atau taruh PC di ruang tamu.
