Ventoy

**Ventoy** adalah *tool* sumber terbuka (open source) yang diciptakan untuk membuat *bootable* USB *drive* yang ajaib. Kenapa ajaib? Karena kamu bisa boot langsung dari file **ISO/WIM/IMG/VHD(x)/EFI**!

**Fokus Utama:** Kamu tidak perlu repot-repot memformat disk berulang kali. Cukup **seret dan lepas** file gambar OS (ISO, dll.) ke USB *drive* kamu, dan *boot*! Kamu bisa menyalin banyak file sekaligus, dan Ventoy akan otomatis menyajikan menu *boot* yang cantik untuk memilih OS mana yang ingin kamu jalankan.

  • Mendukung arsitektur **x86 Legacy BIOS, IA32 UEFI, x86_64 UEFI, ARM64 UEFI, dan MIPS64EL UEFI** secara mulus.
  • Mendukung gaya partisi **MBR dan GPT** dengan cara yang sama.
  • Sudah menguji **1300+ ISO** (Lihat Daftar) dan mendukung **90%+ distro** di DistroWatch.com.

**Website Resmi:** https://www.ventoy.net

Fitur-Fitur Ventoy yang Bikin Hidup Lebih Mudah

Ventoy tidak hanya cepat (hanya dibatasi oleh kecepatan *copy* file ISO kamu), tetapi juga penuh dengan fitur canggih yang dibutuhkan oleh seorang *sysadmin* atau pengoprek serius:

  • **100% Open Source.** Transparan dan bisa diverifikasi.
  • **Auto Installation:** Dukungan *script* instalasi otomatis untuk Windows dan Linux.
  • **Boot Langsung:** Boot langsung dari ISO/WIM/IMG/VHD(x)/EFI tanpa perlu proses ekstraksi.
  • **Dukungan File System Luas:** Mendukung **FAT32/exFAT/NTFS/UDF/XFS/Ext2(3)(4)** untuk partisi utama.
  • **Instalasi Fleksibel:** Bisa dipasang di USB, Local Disk, SSD, NVMe, atau SD Card.
  • **ISO Besar:** Mendukung file ISO yang ukurannya **lebih besar dari 4GB**.
  • **Menu Canggih:** Ada *Menu alias*, *Password protect*, mode *TreeView*, dan *highly customizable theme*.
  • **Nondestructive Upgrade:** Data di USB kamu aman saat upgrade versi Ventoy.
  • **Persistence:** Dukungan Persistence untuk Linux (menyimpan perubahan sesi).
  • **Secure Boot:** Didukung untuk IA32/x86_64 UEFI.
  • **VentoyPlugson:** GUI configurator untuk mengelola *plugin*.

Tampilan Menu Boot UEFI Ventoy

**Ventoy Compatible Concept:** Ventoy punya konsep unik. Selama file kamu kompatibel dengan Ventoy, kamu **tidak perlu mengupdate Ventoy** di USB kamu hanya karena ada distro Linux baru yang dirilis. Cukup *copy* file ISO-nya, dan beres!
